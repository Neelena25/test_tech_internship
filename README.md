# Тестовая работа для QA (Авито)

**Task_1.md** - Задание 1 

**TESTCASES.md** - тест-кейсы для тестирования сервиса

**BUGS.md** - баги, выявленные при тестировании сервиса


Папка **AutoTests** - проект для автотестирования.

Внутри AutoTests находится **.env** с базовым url + папка **tests**.


Содержание **tests**:

Название файла  | Содержание файла
----------------|----------------------
tests_del       | .py файлы c проверкой статуса сервера при удалении
tests_get       | .py файлы с проверкой статуса сервера при получении объявлении по SellerID
tests_get_statistic   | .py файлы с проверкой статуса сервера при получении статистики по SellerID
tests_post       | .py файлы с проверкой статуса сервера при сохранении объявлении
conftest.py       | настраивает инфраструктуру для безопасного и удобного тестирования 


Для тестирования в проект устанавливаются пакеты с помощью:
**Bpip install pytest requests python-dotenv**

## Запуск автотестирования:
В терминале прописать: **pytest -v**
